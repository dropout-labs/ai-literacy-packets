## Encabezado

**Actividad 1: Ordenar por característica.**

## Visual

Una cuadrícula de 12 dibujos pequeños:

```
┌────────┬────────┬────────┬─────────┐
│ plátano│ manzana│ pelota │  pelota  │
│        │        │de béisbol│de básquet│
├────────┼────────┼────────┼─────────┤
│autobús │  libro │mochila │cuaderno │
│escolar │de texto│        │         │
├────────┼────────┼────────┼─────────┤
│calcula-│ pelota │ limón  │ funda   │
│ dora   │de fútbol│        │de teléfono│
└────────┴────────┴────────┴─────────┘
```

Debajo de la cuadrícula, tres cajas:

```
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ Caja 1:          │  │ Caja 2:          │  │ Caja 3:          │
│ Cosas que        │  │ Cosas que        │  │ Cosas que usas   │
│ son redondas     │  │ son amarillas    │  │ en la escuela    │
│                  │  │                  │  │                  │
└──────────────────┘  └──────────────────┘  └──────────────────┘
```

## Instrucciones

> Mira las 12 cosas en la cuadrícula. Cada cosa pertenece a **al menos una** de las tres cajas — y algunas pertenecen a más de una. Dibuja una línea de cada cosa a cada caja a la que pertenece.
>
> Después contesta:
>
> - ¿Cuál cosa terminó en la **mayor cantidad** de cajas?
> - ¿Cuál cosa terminó en la **menor cantidad**?

## Pregunta de discusión (al pie)

> Una computadora solo puede ordenar cosas por **características que puede medir**. Si una computadora solo pudiera ver colores y formas, ¿cuáles de tus tres cajas podría descubrir por sí misma?
>
> ¿Cuál necesita algo que no puede ver?

## Nota al margen

> Una "característica" es solo algo sobre una cosa que puedes medir o verificar. *Redondo* es una característica. *Amarillo* es una característica. *Usado en la escuela* también es una característica — pero depende de saber qué es la escuela.

## Lo que esto enseña

Escoger las características cambia el orden. La IA no decide qué características importan — las personas que la construyen lo deciden.
