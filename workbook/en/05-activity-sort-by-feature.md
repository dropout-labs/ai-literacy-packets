# Spread 05 — Activity: Sort by feature

> Part 1: What is AI? · spread 5 of 16 · Activity A1 · 10 minutes

## Heading

**Activity 1: Sort by feature.**

## Visual

A grid of 12 small line drawings:

```
┌────────┬────────┬────────┬────────┐
│ banana │ apple  │ baseball│ basketball│
├────────┼────────┼────────┼────────┤
│ school │textbook│backpack│notebook│
│  bus   │        │        │        │
├────────┼────────┼────────┼────────┤
│calculator│ soccer│ lemon  │ phone  │
│        │  ball │        │ case   │
└────────┴────────┴────────┴────────┘
```

Below the grid, three boxes:

```
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│ Box 1:           │  │ Box 2:           │  │ Box 3:           │
│ Things that      │  │ Things that      │  │ Things you use   │
│ are round        │  │ are yellow       │  │ at school        │
│                  │  │                  │  │                  │
└──────────────────┘  └──────────────────┘  └──────────────────┘
```

## Instructions

> Look at the 12 things in the grid. Each thing belongs in **at least one** of the three boxes — and some belong in more than one. Draw a line from each thing to every box it fits in.
>
> Then answer:
>
> - Which thing ended up in the **most** boxes?
> - Which thing ended up in the **fewest**?

## Discussion prompt (bottom)

> A computer can only sort things by **features it can measure**. If a computer could only see colors and shapes, which of your three boxes could it figure out by itself?
>
> Which one needs something it can't see?

## Margin note

> A "feature" is just a thing about something that you can measure or check. *Round* is a feature. *Yellow* is a feature. *Used at school* is a feature too — but it depends on knowing what school is.

## What this teaches

Choosing the features changes the sorting. AI doesn't decide what features matter — the people building it do.
