## What this is

DROPOUT Labs is a California 501(c)(3) public benefit nonprofit (EIN 41-5068177) based in Santa Maria. We design and freely distribute **bilingual, take-home AI literacy packets** for youth ages 12–18 in underserved Santa Barbara County communities.

Every packet is:
- **Bilingual** — English on the left page, Spanish on the right, every page
- **Take-home** — works on a kitchen table, no WiFi, no teacher, no enrollment
- **Open-source** — every page released under Creative Commons so any library, school, or sibling nonprofit can reprint forever at cost

This repository is the canonical home of the curriculum. Anything here can be downloaded, printed, remixed, and redistributed freely under the licenses below.

## How to reprint at cost (libraries, schools, partners)

Once v1.0 ships, reprinting is intended to be a 5-minute decision:

1. Download the print-ready PDFs from the latest release
2. Send to your local print shop (or library's risograph / docucenter)
3. Add your logo to the back-cover co-branding slot if desired
4. Distribute

You don't need permission. You don't need to credit DROPOUT Labs (though it's appreciated). You don't owe us a fee. The CC BY-SA 4.0 license is a permanent invitation.

See [`partners/README.md`](partners/README.md) for the full reprint kit (when published).

## Why open-source?

Every grant dollar that funds curriculum here produces an asset that any community in any language can reuse forever. One funded curriculum compounds; one funded program ends when the funding ends. This repo is structurally what makes DROPOUT Labs a sustainable organization rather than a series of one-off pilots.
