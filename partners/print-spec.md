## Print spec

| Field | Value |
|---|---|
| **Trim size** | 8.5" × 11" (US Letter) |
| **Page count (v1.0 PDF)** | 65 pages — *see "Known overflow" below* |
| **Target page count (v1.1)** | 32 pages |
| **Bleed** | 0.125" all sides |
| **Color** | 4-color process (CMYK), throughout |
| **Cover stock** | 80–100 lb. coated cover (matte preferred) |
| **Interior stock** | 60–70 lb. uncoated text (cream or natural white) |
| **Bind** | Saddle-stitch (2 staples on the spine) |
| **Folding** | Standard book fold |
| **Trim** | Standard 3-knife trim, registered to bleed marks |

### Known overflow (read this before quoting)

The current v1.0 print-ready PDF is **65 pages, not 32**, because some content-dense spreads overflow to a second printed page in our generated PDF. We are aware. We are scheduling a designer layout pass (v1.1, est. June–July 2026) to bring the bilingual edition to a true 32-page saddle-stitch booklet.

If quoting now (v1.0): quote against the 65-page PDF as printed.
If quoting for v1.1 (recommended for a real first run): quote against a target 32-page bilingual booklet, same trim and bind.
