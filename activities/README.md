# Activities

Unplugged paper activities that pair with the workbook. Designed to work without WiFi, computers, or a teacher.

## Status

Pre-v0.1 — activity inventory pending. See repo root [README.md](../README.md) for milestones.

## Planned activity types

- **Pattern-finding worksheets** — paper-based machine-learning exercises (sort by feature, predict the next, find the rule)
- **Prompt-design cards** — fold-out cards practicing how to ask AI for what you actually want
- **"Spot the AI" annotation sheets** — analyze real-world examples and circle what's AI-generated
- **Family conversation guide** — discussion prompts for youth and parents/guardians
- **Reflection journal pages** — youth document what they learned and what surprised them

## Bilingual

Every activity ships in English and Spanish.
