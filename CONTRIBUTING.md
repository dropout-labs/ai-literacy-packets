# Contributing to AI Literacy Packets

Thank you for considering a contribution. This curriculum exists for the youth and families it serves, and contributions from translators, educators, illustrators, and library and school partners directly improve their experience.

## Ways to contribute

- **Translation feedback** — Spanish-register validation, dialect concerns, accessibility-of-language notes
- **Activity ideas** — unplugged paper activities, prompt cards, "spot the AI" worksheets
- **Illustration** — visual design, accessible iconography, low-reading-level visual storytelling
- **Reprint reports** — if you (a library, school, sibling nonprofit) reprinted this packet, tell us via an issue using the *Reprint Report* template — we want to know who is using it and how
- **Recipient feedback** — youth and family feedback via the *Recipient Feedback* issue template
- **Code** — improvements to repo tooling, automation, build scripts (LICENSE-CODE / MIT)

## How to contribute

1. Open an issue describing what you want to do (or pick an existing one)
2. For curriculum changes: fork, edit the relevant `workbook/`, `activities/`, or `partners/` files, open a PR
3. For code changes: standard fork → branch → PR flow

## Bilingual policy

This is a bilingual project (English + Spanish). All curriculum content must ship in both languages before release. Code, comments, issue templates, and contributor docs may be English-only for now; future releases will translate these as well. If you can contribute Spanish translations of contributor-facing docs, please do.
